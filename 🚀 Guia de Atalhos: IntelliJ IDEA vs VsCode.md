# 🚀 Guia de Atalhos: IntelliJ IDEA vs VS Code (50 Essenciais)

Este guia compara os principais atalhos para desenvolvedores que transitam entre as duas IDEs mais populares do mercado.

---

## 📂 1. Navegação e Busca
| Função | IntelliJ IDEA | VS Code |
| :--- | :--- | :--- |
| Pesquisar Tudo (Search Everywhere) | `Shift` `Shift` | `Ctrl` + `P` |
| Abrir Arquivo pelo Nome | `Ctrl` + `Shift` + `N` | `Ctrl` + `P` |
| Localizar Texto no Arquivo | `Ctrl` + `F` | `Ctrl` + `F` |
| Localizar Texto no Projeto | `Ctrl` + `Shift` + `F` | `Ctrl` + `Shift` + `F` |
| Substituir no Arquivo | `Ctrl` + `R` | `Ctrl` + `H` |
| Substituir no Projeto | `Ctrl` + `Shift` + `R` | `Ctrl` + `Shift` + `H` |
| Ir para a Linha específica | `Ctrl` + `G` | `Ctrl` + `G` |
| Lista de Arquivos Recentes | `Ctrl` + `E` | `Ctrl` + `Tab` |
| Ir para Símbolo (Método/Variável) | `Ctrl` + `Alt` + `Shift` + `N` | `Ctrl` + `Shift` + `O` |
| Abrir Terminal Integrado | `Alt` + `F12` | `Ctrl` + ` ` (crase) |
